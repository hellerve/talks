# Rack'n'Stack 2026

A talk on how network automation enables sovereign datacenter environments,
making self-hosted infrastructure feel like cloud. The full title is "Von der
Cloud ins Rack -- Wie Netzautomatisierung souveräne RZ-Umgebungen ermöglicht".

Given in German.
